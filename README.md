# face-recognition-experiments

This project is implemented in Jupyter notebooks and is intended to run on Kaggle.

## Environment
- Platform: Kaggle Notebook
- Hardware: GPU (Kaggle provided)

## Dataset
No dataset files are included in this repository.

The datasets are provided by Kaggle and accessed via `/kaggle/input/`.

Links:
- MS1M-ArcFace (Kaggle):  
  https://www.kaggle.com/datasets/yakhyokhuja/ms1m-arcface-dataset  
- Labeled Faces in the Wild (LFW):  
  https://www.kaggle.com/datasets/jessicali9530/lfw-dataset


## Files
- `training.ipynb`: training and validation code
- `testing.ipynb`: testing and evaluation code

## How to Run
1. Create a new Kaggle Notebook.
2. Enable GPU in **Settings → Accelerator → GPU T4×2**.
3. Add the required Kaggle dataset to the notebook.
4. Upload or copy the notebooks from this repository.
5. Run all cells in `training.ipynb`, then `testing.ipynb`.

## Notes
This repository contains code only. All experiments were conducted using Kaggle-provided resources.
